1. Сопротивление двух, последовательно соединенных резисторов равно R = R1+ R2. При последовательном соединении сопротивление больше, чем каждого из резисторов.
2. Сопротивление двух параллельно включенных резисторов равно R = R1R2 / (R1 + R2). Или 1/R = 1/R1 + 1/R2. Общее сопротивление меньше, чем сопротивление каждого из резисторов. 

Общие правила:
1. Если сопротивление одного из двух соединенных последовательно резисторов много больше сопротивления второго, то общее сопротивление будет примерно равно сопротивлению первого резистора.
2. Если сопротивление одного из двух соединенных параллельно резисторов много больше сопротивления второго, то общее сопротивление будет примерно равно сопротивлению второго резистора.
