При транспонировании матрицы строки и столбцы меняются местами, таким образом из матрицы nXm получается матрица mXn:

![[{F0AF580B-F57B-4DAE-97ED-83AFF1E4ECE0}.png]]

Пример реализации на C++:
```
template <typename T>
Matrix<T> Matrix<T>::transpose() const
{
    Matrix<T> out(columns, rows);

    unsigned int i, ii;
    for(i=0; i<columns; i++){
        for(ii=0; ii<rows; ii++){
            out.data[i][ii] = data[ii][i];
        }
    }
  
    return out;
}
```
