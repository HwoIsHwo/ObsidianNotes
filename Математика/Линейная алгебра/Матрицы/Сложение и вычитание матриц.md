Данные операции выполняются при помощи двух матриц, которые должны иметь одинаковый размер, то есть, одинаковое количесвто строк ис столбцов. При выполнении операций элементы на одинаковых позициях складываются/вычитаются.

Пример сложения матриц на C++:
```
template <typename T>
template <typename V>
Matrix<T> Matrix<T>::operator+(const Matrix<V>& m)

{
    if(m.rows!=rows || m.columns!=columns){
        std::cout << "operator +: matrices of different sizes" << std::endl;
        return *this;
    }

    Matrix<T> out(rows, columns);

    unsigned int i, ii;
    for(i=0; i<rows; i++){
        for(ii=0; ii<columns; ii++){
            out.data[i][ii] = data[i][ii] + (m.data[i][ii]);
        }
    }

    return out;
}
```
