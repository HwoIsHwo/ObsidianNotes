Матрица - математический объект, описываемый в виде прямоугольной таблицы с определённым числом столбцов и строк, на пересечении которых находятся элементы матрицы. Матрица n\*m имеет n строк и m столбцов.

==Для матрицы определены следующие алгебраические операции:==
- сложение матриц, имеющих один и тот же размер;
- умножение матриц подходящего размера (матрицу, имеющую  n столбцов, можно умножить справа на матрицу, имеющую n строк);
- в том числе умножение матрицы на вектор-столбец и умножение вектор-строки на матрицу (по обычному правилу матричного умножения; вектор является в этом смысле частным случаем матрицы);
- умножение матрицы на скаляр.

Пример объявления матрицы на C++:
```
class Matrix

{

public:

    unsigned int rows; //строки матрицы

    unsigned int columns; //столбцы матрицы

    double* data; //указатель на данные матрицы

};
```
