Для данной операции используются две матрицы, при этом количество столбцов левой матрицы должно совпадать с количеством строк правой матрицы. Допустим, левая матрицы mXn, тогда правая - nXk. Результат операции - матрица mXk. Каждый элемент результирующей матрицы определяется как сумма произведений соответствующих элементов строки левой матрицы на столбец правой матрицы.

Пример реализации на C++:
```
template <typename T>
template <typename V>
Matrix<T> Matrix<T>::operator*(const Matrix<V>& m)
{
    if(columns!=m.rows){
        std::cout << "operator *: number of columns of left operand != number of rows of right operand" << std::endl;
        return *this;
    }

    Matrix<T> out(rows, m.columns);

    unsigned int i, ii, iii;
    T result = 0;
    for(i=0; i<out.rows; i++){
        for(ii=0; ii<out.columns; ii++){
            for(iii=0; iii<columns; iii++){
                result += data[i][iii] * m.data[iii][ii];
            }
            out.data[i][ii] = result;
            result = 0;
        }
    }

    return out;
}
```
