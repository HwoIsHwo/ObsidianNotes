Образ - готовое решение, содержащее определенный функционал. Образ неизменяем. На основе образа можно создать контейнер.
Проект с кодом, образами, файлами - контейнер.
То есть, внутри проекта может быть реализован код на различных языках программирования с образом для каждого из них и все это будет работать внутри одного контейнера.

Образы получают с <u>dockerhub</u> при помощи команды <u>pull</u>.
Для запуска образа в интерактивном режиме используется опция -it. Также при запуске образа каждый раз создается новый контейнер. Для задания имени контейнеру используется опция --name.
