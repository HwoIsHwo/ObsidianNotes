Бинарное дерево — это дерево, в котором у каждого узла не более двух потомков. Обычно их называют левым и правым. Структура может быть произвольной (без упорядочения), а может обладать дополнительным свойством, как в бинарном дереве поиска (BST), где для каждого узла выполняется правило: все значения в левом поддереве меньше текущего, в правом — больше.

Если дерево поиска сбалансировано, операции поиска, вставки и удаления выполняются за O(log n). Если нет — возможна деградация до O(n), например при вставке уже отсортированной последовательности.

Минимальная реализация обычного бинарного дерева поиска на C может выглядеть так:
```C
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int value;
    struct Node* left;
    struct Node* right;
} Node;

Node* create_node(int value) {
    Node* n = malloc(sizeof(Node));
    if (!n) return NULL;
    n->value = value;
    n->left = NULL;
    n->right = NULL;
    return n;
}

Node* insert(Node* root, int value) {
    if (root == NULL)
        return create_node(value);

    if (value < root->value)
        root->left = insert(root->left, value);
    else if (value > root->value)
        root->right = insert(root->right, value);

    return root;
}

Node* find(Node* root, int value) {
    if (root == NULL || root->value == value)
        return root;

    if (value < root->value)
        return find(root->left, value);
    else
        return find(root->right, value);
}

void inorder(Node* root) {
    if (!root) return;

    inorder(root->left);
    printf("%d ", root->value);
    inorder(root->right);
}

void free_tree(Node* root) {
    if (!root) return;
    free_tree(root->left);
    free_tree(root->right);
    free(root);
}

int main() {
    Node* root = NULL;

    root = insert(root, 5);
    insert(root, 2);
    insert(root, 8);
    insert(root, 1);
    insert(root, 3);

    inorder(root);  // 1 2 3 5 8
    printf("\n");

    Node* n = find(root, 3);
    if (n) printf("Found: %d\n", n->value);

    free_tree(root);
    return 0;
}
```

Здесь используется рекурсивная вставка. Обход `inorder` для BST выводит элементы в отсортированном порядке. Управление памятью полностью ручное.

В стандартной библиотеке C++ нет контейнера с именем «binary_tree», но ассоциативные контейнеры реализованы на основе сбалансированного бинарного дерева (обычно красно-чёрного). Это `std::set`, `std::map`, `std::multiset`, `std::multimap`.

Пример с `std::set`:
```C
#include <iostream>
#include <set>

int main() {
    std::set<int> s;

    s.insert(5);
    s.insert(2);
    s.insert(8);
    s.insert(1);
    s.insert(3);

    for (int v : s)
        std::cout << v << " ";
    std::cout << "\n";

    if (s.find(3) != s.end())
        std::cout << "Found 3\n";

    return 0;
}
```
`std::set` гарантирует логарифмическую сложность операций за счёт балансировки дерева, в отличие от примитивного BST на C. Конкретная реализация (красно-чёрное дерево) стандартизирована только по свойствам сложности, но не по точной структуре.
