Бинарное дерево поиска (Binary Search Tree, BST) — это двоичное дерево, в котором для каждой вершины выполняется упорядоченность ключей: все элементы в левом поддереве меньше ключа текущего узла, все элементы в правом — больше. Это локальное правило рекурсивно выполняется для всех поддеревьев.

Фактически BST — это структура, позволяющая реализовать ассоциативный контейнер без необходимости линейного поиска. Средняя сложность основных операций (`insert`, `find`, `erase`) равна `O(log N)`, но только при условии близкой к сбалансированной структуры. В вырожденном случае (например, вставка уже отсортированных данных) дерево превращается в список, и сложность деградирует до `O(N)`.

Ключевая особенность BST — операции всегда выполняются сравнением и переходом либо влево, либо вправо. Никаких сдвигов памяти, как в массиве, не требуется.


## Минимальная реализация BST на C
Структура узла максимально простая — хранит значение и два указателя.
```C
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int key;
    struct Node* left;
    struct Node* right;
} Node;

Node* create_node(int key)
{
    Node* n = malloc(sizeof(Node));
    n->key = key;
    n->left = NULL;
    n->right = NULL;
    return n;
}

Node* insert(Node* root, int key)
{
    if (root == NULL)
        return create_node(key);

    if (key < root->key)
        root->left = insert(root->left, key);
    else if (key > root->key)
        root->right = insert(root->right, key);

    return root;
}

Node* find(Node* root, int key)
{
    if (root == NULL || root->key == key)
        return root;

    if (key < root->key)
        return find(root->left, key);

    return find(root->right, key);
}

void inorder(Node* root)
{
    if (!root) return;

    inorder(root->left);
    printf("%d ", root->key);
    inorder(root->right);
}



int main()
{
    Node* root = NULL;

    root = insert(root, 5);
    insert(root, 2);
    insert(root, 8);
    insert(root, 1);
    insert(root, 3);

    inorder(root); // 1 2 3 5 8

    Node* n = find(root, 3);
    if (n)
        printf("\nFound: %d\n", n->key);
}
```


## BST в C++ STL
В STL **нет контейнера с названием BST**, но несколько контейнеров реализованы поверх **красно-чёрного дерева (Red-Black Tree)** — это сбалансированный вариант BST.
К ним относятся:
- `std::set`
- `std::multiset`
- `std::map`
- `std::multimap`

То есть фактически STL уже даёт промышленную реализацию BST.
```C++
#include <set>
#include <iostream>

int main()
{
    std::set<int> tree;

    tree.insert(5);
    tree.insert(2);
    tree.insert(8);
    tree.insert(1);

    for (int v : tree)
        std::cout << v << " ";
}
```
Вывод:
1 2 5 8
Элементы автоматически хранятся упорядоченно.

Поиск:
```C++
auto it = tree.find(2);

if (it != tree.end())
    std::cout << "found\n";
```
Все операции гарантированно имеют сложность `O(log N)` благодаря балансировке.
