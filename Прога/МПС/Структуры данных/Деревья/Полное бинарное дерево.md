Полное бинарное дерево (Full Binary Tree, FBT) — это бинарное дерево, в котором **каждая внутренняя вершина имеет ровно двух потомков**. Узел либо лист, либо имеет двух детей. Узлов с одним ребёнком быть не может.

Это важно не путать с _complete binary tree_ (почти полным деревом), которое обычно используется в кучах. В терминологии часто возникает путаница:
- **Full Binary Tree (FBT)** — 0 или 2 потомка.
- **Complete Binary Tree** — уровни заполнены слева направо.
- **Perfect Binary Tree** — все уровни полностью заполнены.

Для FBT выполняется полезное свойство. Если число внутренних узлов равно `I`, тогда число листьев: L = I + 1
а общее число узлов: N = 2 * I + 1
Отсюда следует, что число узлов всегда нечётное.


### Минимальная реализация на C
FBT — это не отдельная структура хранения, а **ограничение на построение обычного бинарного дерева**. Поэтому реализация сводится к обычному узлу и контролю вставки.
Минимальный вариант:
```C
#include <stdio.h>
#include <stdlib.h>

typedef struct Node
{
    int value;
    struct Node* left;
    struct Node* right;
} Node;

/* создание листа */
Node* create_leaf(int value)
{
    Node* n = (Node*)malloc(sizeof(Node));
    n->value = value;
    n->left = NULL;
    n->right = NULL;
    return n;
}

/* создание внутреннего узла (обязательно 2 ребенка) */
Node* create_parent(int value, Node* left, Node* right)
{
    if (!left || !right)
    {
        fprintf(stderr, "FBT violation: parent must have 2 children\n");
        exit(EXIT_FAILURE);
    }

    Node* n = (Node*)malloc(sizeof(Node));
    n->value = value;
    n->left = left;
    n->right = right;

    return n;
}

void inorder(Node* root)
{
    if (!root) return;

    inorder(root->left);
    printf("%d ", root->value);
    inorder(root->right);
}

void free_tree(Node* root)
{
    if (!root) return;

    free_tree(root->left);
    free_tree(root->right);
    free(root);
}

int main()
{
    Node* l1 = create_leaf(1);
    Node* l2 = create_leaf(2);
    Node* l3 = create_leaf(3);
    Node* l4 = create_leaf(4);

    Node* p1 = create_parent(5, l1, l2);
    Node* p2 = create_parent(6, l3, l4);

    Node* root = create_parent(7, p1, p2);

    inorder(root);
    free_tree(root);
}
```

Ключевой момент: функция `create_parent()` запрещает появление одного потомка. Тем самым инвариант FBT поддерживается конструктивно.


В стандартной библиотеке C++ **нет структуры “binary tree” как контейнера**. STL предоставляет только структуры, построенные _поверх_ деревьев:
- `std::set`
- `std::map`
- `std::multiset`
- `std::multimap`

Они реализованы как **красно-чёрные деревья** (self-balancing BST), а не как FBT.
Красно-чёрное дерево не является full binary tree, потому что узел может иметь одного ребёнка.
Пример использования STL-дерева:
```C++
#include <set>
#include <iostream>

int main()
{
    std::set<int> tree;

    tree.insert(5);
    tree.insert(1);
    tree.insert(9);
    tree.insert(3);

    for (int v : tree)
        std::cout << v << " ";
}
```
