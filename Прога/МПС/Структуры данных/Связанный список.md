Связанный список — это структура данных, состоящая из узлов, где каждый узел хранит данные и ссылку (или несколько ссылок) на другие узлы. В классическом односвязном списке каждый элемент содержит указатель на следующий. В двусвязном — ещё и на предыдущий.

Ключевая особенность — элементы не обязаны лежать в памяти подряд. Каждый узел выделяется отдельно (обычно в куче), а связь обеспечивается через указатели. В отличие от массива, где элементы расположены непрерывно и индекс вычисляется арифметически.

Когда список применим. Он удобен, если:  
– требуется частая вставка и удаление элементов в середине структуры;  
– размер заранее неизвестен и сильно меняется;  
– не требуется быстрый произвольный доступ по индексу.

Вставка в середину связанного списка при наличии указателя на позицию выполняется за O(1): нужно лишь перенастроить пару ссылок. В массиве это O(n), потому что требуется сдвиг элементов.

Но у списка есть существенные недостатки. Нет произвольного доступа — чтобы получить i-й элемент, нужно пройти список от начала за O(n). Память используется менее эффективно: каждый узел хранит дополнительный указатель, плюс возможны накладные расходы аллокатора. Кэш-локальность хуже — элементы разбросаны по памяти, что замедляет последовательный проход по сравнению с массивом.

В современных CPU это критично: линейный проход по массиву часто быстрее, чем по списку, даже если асимптотическая сложность одинаковая. Поэтому связанный список имеет смысл только там, где действительно доминируют вставки/удаления при уже известной позиции.

Минимальная реализация односвязного списка на C:
```C
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int value;
    struct Node* next;
} Node;

void push_front(Node** head, int value)
{
    Node* new_node = malloc(sizeof(Node));
    new_node->value = value;
    new_node->next = *head;
    *head = new_node;
}

void print_list(Node* head)
{
    while (head) {
        printf("%d ", head->value);
        head = head->next;
    }
    printf("\n");
}

void free_list(Node* head)
{
    while (head) {
        Node* tmp = head;
        head = head->next;
        free(tmp);
    }
}

int main()
{
    Node* head = NULL;

    push_front(&head, 3);
    push_front(&head, 2);
    push_front(&head, 1);

    print_list(head);  // 1 2 3

    free_list(head);
    return 0;
}
```


Это минимальный пример: вставка в начало и линейный проход. Вставка в середину требует поиска предыдущего узла.

В C++ стандартная библиотека предоставляет готовую реализацию — std::list. Это двусвязный список.

Пример использования:
```C++
#include <list>
#include <iostream>

int main()
{
    std::list<int> lst;

    lst.push_back(1);
    lst.push_back(2);
    lst.push_front(0);

    for (int v : lst)
        std::cout << v << " ";

    std::cout << "\n";

    auto it = lst.begin();
    ++it;
    lst.insert(it, 42);  // вставка в середину

    for (int v : lst)
        std::cout << v << " ";
}
```

Важно понимать, что `std::list` не поддерживает оператор `[]`, потому что нет произвольного доступа. Итераторы — двунаправленные, но не random-access.

С практической точки зрения в большинстве прикладных задач массив (`std::vector`) оказывается быстрее и проще. Связанный список оправдан в узких сценариях: сложные операции сплайсинга, частые перемещения элементов без копирования, реализация специализированных структур (например, intrusive lists в системном коде).
