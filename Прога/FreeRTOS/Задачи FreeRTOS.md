## Упреждающее планирование по приоритету (без разделения по времени)
Если распределение времени между задачами одного приоритета не используется, то планировщик переключает задачу только в следующих случаях:
1. Задача с более высоким приоритетом переходит в состояние готовности
2. Выполняемая задача попадает в блокировку

## Совместное планирование (без разделения по времени и упреждения)
Переключение задач происходит только когда задача сообщает планировщику, что она готова отдать управление. То есть контекст переключается когда задача переходит из состояния выполнения в состояние блокировки.
