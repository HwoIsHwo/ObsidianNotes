Линейная регрессия — один из основных и широко используемых алгоритмов машинного обучения. Алгоритмы линейной регрессии вычисляют выходную переменную (зависимую переменную) как взвешенную комбинацию одной или нескольких входных переменных (независимых переменных). Определяя оптимальный вес каждой входной переменной, алгоритм выводит «линию наилучшего соответствия» для точек данных, полученных в процессе обучения. Существует множество вариантов и расширений линейной регрессии, использующих схожую логику для моделирования более сложных математических взаимосвязей между зависимыми и независимыми переменными, например, полиномиальная регрессия (моделирующая нелинейные, криволинейные взаимосвязи) или квантильная регрессия (моделирующая взаимосвязи в определённых точках распределения набора данных).

Алгоритмы дерева решений формируют окончательные прогнозы выходных данных посредством ветвящейся последовательности решений типа «если-то-иначе», которую можно визуализировать в виде древовидной диаграммы. Их можно использовать как для регрессии, так и для классификации. В отличие от большинства алгоритмов обучения с учителем, цель алгоритма заключается не в оптимизации выходных предсказаний путём минимизации одной глобальной функции потерь, а в оптимизации предсказательной силы каждого отдельного узла ветвления дерева.

Модели пространства состояний (SSM) моделируют динамические системы и последовательные данные с помощью двух взаимосвязанных уравнений: одно, уравнение состояния, описывает внутреннюю динамику («состояние») системы, которая не поддаётся непосредственному наблюдению; другое, уравнение выхода, описывает, как эта внутренняя динамика соотносится с наблюдаемыми результатами, то есть с выходными данными системы. Они используются в самых разных областях: от электротехники до финансового прогнозирования и обработки естественного языка (NLP).
