В глубоком обучении в качестве моделей используются нейросети с большим количеством слоев (глубокие), а не традиционные алгоритмы машинного обучения.

Нейросети состоят из отдельных слоев, каждый слой - из нейронов. Каждый нейрон выполняет свою математическую операцию, называемую функцией активации. Выход функции активации (нейрона) служит входом для всех нейронов следующего слоя (и так далее, до самого выхода нейросети). Обычно функции активации являются нелинейными функциями, что позволяет анализировать сложные зависимости данных.

![[Pasted image 20251110202146.png]]

Каждое соединение между нейронами имеет уникальное значение - вес - множитель, который влияет на воздействие нейрона на вход нейрона следующего слоя. Эти веса, наряду с уникальными членами - смещениями, добавленными к функции активации каждого нейрона, являются параметрами, которые оптимизируются с помощью машинного обучения.

Один из основных алгоритмов машинного обучения, алгоритм обратного распространения, позволяет вычислять, как каждый отдельный узел вносит вклад в общий вывод функции потерь, позволяя индивидуально оптимизировать даже миллионы или миллиарды весов модели с помощью алгоритмов градиентного спуска.

Такая многослойная структура дает моделям глубокого обучения невероятную гибкость и универсальность. ==Нейронные сети являются универсальными аппроксиматорами: теоретически доказано, что для любой функции существует конфигурация нейронной сети, способная её воспроизвести.==
