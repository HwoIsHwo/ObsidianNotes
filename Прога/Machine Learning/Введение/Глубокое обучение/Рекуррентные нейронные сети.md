Рекуррентные нейронные сети (РНС) предназначены для работы с последовательными данными. В то время как обычные нейронные сети прямого распространения сопоставляют один входной сигнал с одним выходным сигналом, RNN сопоставляют последовательность входных данных с выходным сигналом, работая в рекуррентном цикле, в котором выходной сигнал для заданного шага входной последовательности служит входным сигналом для вычислений на следующем шаге. По сути, это создает внутреннюю «память», называемую скрытым состоянием, которая позволяет RNN понимать контекст и порядок.
