==Машинное обучение специализируется на создании программ, способных менять свое поведение в зависимости от входных данных.== Если в случае задачи может потребоваться большой и сложный код, явно решающий эту задачу, то в случае машинного обучения программа изначально не приспособлена к решению задачи, но по мере увеличения объема данных, используемых для обучения, поведение программы становится все ближе к необходимому. При этом внутри программы не прописан явно алгоритм решения поставленной задачи.
Процесс улучшения качества работы программы путем обработки ей массивов данных и называется обучением. В конце обучения программа умеет сама делать прогнозы и принимать решения без явных инструкций в коде, лишь за счет значений входных данных.

Машинное обучение это подобласть искусственного интеллекта (ИИ), сфокусированная на алгоритмах, которые могут "заучивать" паттерны в данных и, в последствии, делать точные выводы на основе новых данных. Но при этом эта возможность алгоритмов делать предсказания и прогнозы, не заложена внутри них явными инструкциями.

Центральная цель машинного обучения - оптимизация поведения программы на тренировочном наборе валидных данных (обучение), схожих с теми, на которых она будет использоваться. После удачного завершения обучения программа будет способна выдавать правильный результат с использованием новых данных, которых даже не было в тренировочном наборе. На самом деле обучение - всего лишь инструмент. ==**Фундаментальная цель машинного обучения - обобщение тренировочных данных, для получения результатов на реальных.**==

==Глубокое обучение - одна из областей машинного обучения==. Она основана на больших, "глубоких" алгоритмах, называемых искусственными нейронными сетями (ИНС). По сути ИНС - сети из множества математических операций. Такая структура алгоритма позволяет показывать хорошие результаты на довольно сложных данных.

Машинное обучение тесно связанно с анализом данных (DataScience). В каком-то смысле машинное обучение - набор алгоритмов и операций над данными для автоматизации анализа данных и применения результатов этого анализа для задач в реальной жизни.
