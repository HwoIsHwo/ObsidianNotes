В случае обучения с учителем разметка (маркировка) данных может быть очень трудоемкой для сложных задач и больших наборов данных. В самообучении же используются неразмеченные данные. 

Например, автокодировщики обучаются сжимать данные, а затем восстанавливать, используя сжатое представление. Цель обучения - минимизировать ошибку реконструкции данных, используя изначальные данные в качестве эталона:

исходные данные --> сжатые моделью --> восстановленные моделью

Самообучение часто ассоциируется с предварительным обучением, поскольку оно может предоставить базовым моделям широкие возможности, которые затем будут доработаны для более конкретных задач.
