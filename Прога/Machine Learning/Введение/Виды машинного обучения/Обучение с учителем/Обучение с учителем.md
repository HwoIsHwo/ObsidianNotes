Такой вид обучения обычно применяется для задач, требующих точности, например, классификация и регрессия. Данный вид обучения подходит как для глубокого обучения, так и для традиционных моделей машинного обучения.

- регрессионные модели применяются для предсказывания непрерывного значения (цена, температура). Примерами могут являться линейная регрессия, полиноминальная регрессия, модели пространства состояний.
- модели классификации предсказывают дискретные данные, определяют к какому классу принадлежит входной набор данных. Примеры: опорные векторы, Байесовский алгоритм, логистическая регрессия.
- алгоритмы машинного обучения с учителем могут использоваться для любой из этих задач. Например входные данные для линейной регрессии могут использоваться для классификации.

Для обучения модели используются правильные "истинные" данные, с которыми сравнивается выход модели. Например в случае распознавания спама, каждое письмо в наборе помечено (СПАМ/НЕ СПАМ). Цель обучения - скорректировать параметры модели, пока выход модели не будет соответствовать истинным данным.

==Правильность работы модели оценивается при помощи функции потерь.== Она измеряет расхождение между выходом модели и правильными данными. Таким образом обучение модели сходится к минимизации функции потерь. Также значение функции потерь используется алгоритмами обучения для оптимизации параметров модели.

В итоге данный вид обучения требует участия человека как минимум для составления правильных наборов данных. Современное МО использует такие термины как "учитель/контроль" и "контрольные сигналы/сигналы учителя" для обозначения источника истинной информации.
