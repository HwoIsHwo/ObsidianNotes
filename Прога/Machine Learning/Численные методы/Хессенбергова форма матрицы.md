Хессенбергова форма матрицы — это «почти» треугольная квадратная матрица, у которой все элементы ниже первой поддиагонали равны нулю (для верхней формы). Она используется для упрощения вычислений, особенно при поиске собственных значений, так как такие матрицы имеют те же собственные значения, что и исходные, но требуют меньше расчетов для их нахождения, например, в QR-алгоритме.

Основным методом приведения матрицы к виду Хессенберга является метод Хаусхолдера. Метод Хаусхолдера основан на использовании ортогональных отражений для преобразования векторов и матриц к более простой форме без искажения их спектральных и нормовых свойств. В его основе лежит идея заменить последовательность локальных элементарных преобразований одним глобальным ортогональным отражением, которое действует сразу на весь вектор или подматрицу.
Ключевым объектом метода является отражение Хаусхолдера — линейное преобразование, представляющее собой отражение относительно гиперплоскости в евклидовом пространстве. Это отражение строится так, чтобы перевести произвольный вектор в вектор, коллинеарный выбранной координатной оси, обычно первой. В результате все остальные компоненты вектора автоматически зануляются, что и используется в численных алгоритмах.
При применении к матрицам отражения Хаусхолдера последовательно действуют на столбцы или строки, приводя матрицу к верхнетреугольному, хессенбергову или трёхдиагональному виду. Поскольку отражения являются ортогональными, такие преобразования не изменяют собственные значения матрицы и обладают высокой численной устойчивостью, что делает метод Хаусхолдера базовым инструментом в современных алгоритмах QR-разложения и вычисления спектра.

Данный метод позволяет занулять компоненты вектора под главной диагональю матрицы (векторы-столбцы), чтобы привести матрицу к форме Хессенберга. При этом собственные числа матрицы сохраняются.


## Алгоритм:
```
    if(m.columns != m.rows){
        std::cout << "hessenberg(): matrix must be square" << std::endl;
        return m;
    }

    if(m.rows < 3) return m;

    Matrix<double> x, v, P, Q;
    Matrix<double> A0{m};
    double norm_x = 0;
    unsigned int i, ii, iii;
    for(i=0; i<(A0.columns-2); i++){
        x = A0.getColumns(i, i).getRows(i+1, A0.rows-1); //вычленяем подстолбец под главной диагональю

        //Евклидова норма вектора x
        norm_x = 0;
        for(ii=0; ii<x.rows; ii++){
            norm_x += x.data[ii][0] * x.data[ii][0];
        }
        norm_x = sqrt(norm_x);

        //Вектор Хаусхолдера
        if(x.data[0][0] >= 0) v = x + x.identity() * norm_x;
        else v = x - x.identity() * norm_x;

        //Матрица отражения
        P = Matrix<double>(x.rows, x.rows).identity() - (v*v.transpose() / (v.transpose()*v).data[0][0] * 2);

        //Встраиваем матрицу отражения в полную матрицу
        Q = Matrix<double>(A0.rows, A0.columns).identity();
        for(ii=0; ii<P.rows; ii++){
            for(iii=0; iii<P.columns; iii++){
                Q.data[ii+(Q.rows-P.rows)][iii+(Q.columns-P.columns)] = P.data[ii][iii];
            }
        }

        //Зануляем выбранный подстолбец
        A0 = Q.transpose() * A0 * Q;
    }

    A0.clean();

    return A0;
```
