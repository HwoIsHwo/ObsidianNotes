`std::list` — двусвязный список. Каждый элемент лежит в отдельном узле, узлы связаны указателями. Непрерывности в памяти нет.
```
std::list<int> lst = {1, 2, 3};
```

Ключевые свойства:
- вставка и удаление по итератору — O(1);
- итераторы и ссылки стабильны (не инвалидируются при вставках);
- нет доступа по индексу;
- хуже кэш-локальность, чем у `vector`.

Основные методы:
```
lst.begin();
lst.end();
lst.insert(it, value);
lst.erase(it);
lst.splice(pos, other, it);

std::list<int> lst = {1, 2, 3};
auto it = lst.begin();
++it; // указывает на 2
lst.insert(it, 10); // {1, 10, 2, 3}
```



## Итераторы
Внутри std::list объявлен тип iteranor:
```
using It = std::list<std::pair<int,int>>::iterator;
```
По сути это объект, который:
- указывает на конкретный узел списка;
- умеет перемещаться `++` и `--`;
- умеет разыменовываться.

Базовые операции:
```
It it = lst.begin();

++it;   // перейти к следующему узлу
--it;   // к предыдущему

(*it).first;
(*it).second;

// или короче
it->first;
it->second;
```

- lst.begin() Возвращает итератор на **первый элемент списка**.
```
auto it = lst.begin();
*it;   // 1

[1, 2, 3, 4]
 ^
 begin()
```

- lst.end() Возвращает итератор **за последним элементом**. Он не указывает ни на один элемент и не подлежит разыменованию.
```
auto it = lst.end();
// *it;  // UB

[1, 2, 3, 4]
            ^
           end()
```

- lst.insert(it, value) Вставляет `value` **перед элементом, на который указывает `it`**.  Возвращает итератор на вставленный элемент.
```
auto it = lst.begin();
++it; // it -> 2

auto new_it = lst.insert(it, 10);

[1, 10, 2, 3, 4]
     ^
   new_it
```

- lst.erase(it) Удаляет элемент, на который указывает `it`.  Возвращает итератор на **следующий элемент**.
```
auto it = lst.begin();
++it; // it -> 2

auto next = lst.erase(it);

[1, 3, 4]
     ^
   next
```

lst.splice(pos, other, it) Перемещает элемент из `other`, на который указывает `it`, в список `lst` **перед `pos`**.
```
std::list<int> lst   = {1, 2, 3};
std::list<int> other = {10, 20, 30};

auto pos = lst.begin();
++pos; // pos -> 2

auto it = other.begin();
++it; // it -> 20

lst.splice(pos, other, it);


[1, 20, 2, 3]
     ^
[10, 30]
```
