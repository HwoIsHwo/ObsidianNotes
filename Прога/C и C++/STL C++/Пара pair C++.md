`std::pair` — простой агрегат из двух объектов. Семантически это «связка из двух значений», без логики и инвариантов.
```
std::pair<int, double> p{1, 3.14};
```

У него есть два публичных поля: p.first; p.second;

Основные операции:
```
std::pair<int, int> a{1, 2};
std::pair<int, int> b{1, 3};

a == b;   // false
a < b;    // true (лексикографически: сначала first, потом second)

auto p = std::make_pair(10, 20);
```
С C++17 чаще используют структурную инициализацию:
```
auto [k, v] = p;
```
