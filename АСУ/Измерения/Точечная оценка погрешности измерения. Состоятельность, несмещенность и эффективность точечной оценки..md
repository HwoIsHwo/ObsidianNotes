Точечная оценка погрешности измерения – это число, которое используется для приближенного определения величины ошибки при измерении. Состоятельность, несмещенность и эффективность являются важными свойствами точечных оценок. Состоятельная оценка сходится к истинному значению параметра по мере увеличения объема выборки. Несмещенная оценка имеет математическое ожидание, равное истинному значению параметра. Эффективная оценка имеет наименьшую дисперсию среди всех несмещенных оценок. 

Свойства точечных оценок:      
+ Состоятельность: Точечная оценка параметра $\theta$ называется состоятельной, если при увеличении объема выборки (числа измерений) n вероятность того, что значение оценки будет далеко от истинного значения параметра, стремится к нулю:
![[Pasted image 20250806083832.png]]
+ Несмещенность: Точечная оценка параметра $\theta$ называется несмещенной, если ее математическое ожидание равно истинному значению параметра: ![[Pasted image 20250806084032.png]]
+ Эффективность: Несмещенная оценка $\theta$ называется эффективной, если она обладает минимальной дисперсией среди всех несмещенных оценок того же параметра. Эффективность показывает, насколько тесно оценки группируются вокруг своего математического ожидания. Чем меньше дисперсия, тем выше эффективность оценки.
