Этапы синтеза:
1. Составление функциональной схемы САУ из функционально необходимых элементов (ТОУ - технологический объект управления, РО - регулирующий орган, ИМ - исполнительный механизм, регулятор, ИУ - измерительное устройство, ЗУ - задающее устройство). Обычно составляется исходя из функциональной схемы автоматизации ФСА.
![[Pasted image 20250709114649.png]]

2. На основании функциональной схемы проектируемой САУ разрабатывается структурная схема, показывающая из каких структурных элементов САУ состоит, и как они связаны между собой. 
![[Pasted image 20250709114911.png]]

3. Математическое описание элементов системы - представление их соответствующими динамическими звеньями. Математическое описание может строиться исходя из результатов идентификации объекта, путем получения его реакции на тестовые входные воздействия и аппроксимации кривой разгона подходящим динамическим звеном. Либо мат. описание строится аналитически, исходя из законов физики: сохранение массы/импульса/энергии, уравнений электрических цепей и другое.

4. Далее САУ представляется как совокупность обобщенного объекта управления и регулятора. (ООУ - ТОУ и все элементы кроме регулятора)
![[Pasted image 20250709115610.png]]

5. Выбор закона регулирования с использованием определенного подхода (минимизация интегральной ошибки. Согласно Орлову: безынерционный объект – П, одноемкостной – ПИ, двухемкостной и одноемкостной с запаздыванием – ПИД, далее с увеличением емкостности объекта – ПИДД, ПИДДД и т.д.). Главным критерием выбора закона регулирования является обеспечение наилучших показателей качества проектируемой САУ

6. Расчет параметров регулятора (метод оптимального модуля, апериодической устойчивости). Также параметры регулятора можно рассчитать так, чтобы выполнялись условия: условие абсолютной управляемости: 
   + ПФ замкнутой САУ по задающему воздействию равна единице: $W_{xg}(s) = 1$;
   + Условие абсолютной инвариантности по k-му возмущению: ПФ замкнутой САУ по k-му возмущающему воздействию равна нулю: $W_{xf}(s) = 0$;

7. Математическое описание замкнутой САУ: находит ПФ по задающему и возмущающему(-им) воздействиям.

8. Получение переходных процессов разработанной САУ

9. Сравнение численных значений показателей качества спроектированной САУ с требуемыми по техническому заданию показателями качества

10. Вывод о соответствии спроектированной САУ техническому заданию
